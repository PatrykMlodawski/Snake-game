# Snake-game

The Snake Game. You can controll with wsad or arrows. Live: https://vigorous-bohr-223827.netlify.app/

## Getting Started

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```
  
### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/PatrykMlodawski/Snake-game.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start app
   ```sh
   npm start
   ```
